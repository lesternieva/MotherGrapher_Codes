# MotherGrapher_Codes
This compilation of Python/SageMath codes employed for the FCO of the group: MotherGrapher (BSM CS 3A).
# Mathematical Modelling FCO - BSM CS 3A
# Project Overview
This projecr exposes the graoh theoretic concepts in application real-life situation namely, traffic sigmal timing. The Conflict Graph - Maximal Independent Set, and Flow Network - Max-Flow/Min-Cut Theorem is applied utilizing the algorithms and codes included. 
## You may refer to SageMathInstallationGuide file to run the codes.

# Folders/Files
Images - This folder contains all the generated images from SageMath. It also contains necessary visuals od the traffic.

conflictGraph.ipynb - This script contains the greedy algorithm for finding the maximal independent set in a conflict graph.

Ford-Fulkerson.ipynb - This script contains the algorithm used for finding the maximum flow and bottleneck edges of a flow network.

data.csv - contains the raw data collected manually with the use of video recording on the field
